how to run datafruits12
---------------------------
1. install node.js :dizzy_face:
2. install bower
```
npm install -g bower
```
3. run npm install in this directory
```
npm install
```
4. run bower install in this directory
```
bower install
```
5. run `gulp server`

Server should be up at http://localhost:9000, gulp will open your browser
automatically.
